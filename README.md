# Make Pong with Lua and Love

## Run it

### Linux

1. Make sure to have Love installed !(love2d)[love2d.org]

2. Clone repository

	git clone https://github.com/Zeljko-Predjeskovic/LuaPong.git

3. Go into project 

	cd LuaPong

4. Run it

	Love .

### Windows

Install Love2d and drag main.lua into the exe file
